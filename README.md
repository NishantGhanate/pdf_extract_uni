# PDF text extract

## System used to develop
```
* Windows 11
* Python 3.10
```

### Project setup 
```
Create virtual env
> python -m venv venv

Activate env on Windows:
> venv\Scripts\activate

Activate env on Linux :
> venv/bin/activate

Install packages
> pip intall -r requirements.txt
```

### Run
```

> python main.py -h

> python main.py --input_pdf "archivo policia unif.pdf" --start_page 233 --end_page 234

```
